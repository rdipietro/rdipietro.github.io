Research


Machine Learning

As a grad student at Hopkins, I work on machine-learning methods and their applications to medicine.

For example, suppose we are given a few endoscopic videos along with labeled surgery phases (preparation, incision, etc.). Can we develop software that learns from these examples and can automatically recognize these phases in future endoscopic videos?

If so, what other components do we need to achieve more long-term goals such as semi-automatic report generation and semi-automatic surgeon-skill evaluation?

Robot Path Planning with Potential Fields

The blue sphere automatically finds its way to the orange position without hitting any obstacles. This was done by iteratively decreasing a potential-energy function that attracts the sphere to its goal position and repels the sphere from each obstacle. (The visualization was created using ROS; I wrote the planning algorithm in C++ to work with a ROS skeleton provided by Dr. Simon Leonard.)

Robot Path Planning with Rapidly-Exploring Random Trees

The Kuka Lightweight Robot automatically finds its way to the goal position without hitting any obstacles. This was done using rapidly-exploring random trees: in the robot's 7-dimensional configuration space, a tree is repeatedly grown to reach random, collision-free configurations until a path to the goal is found. (The visualization was created using ROS; I wrote the planning algorithm in C++ to work with a ROS skeleton provided by Dr. Simon Leonard.)
